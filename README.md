# 💳 Fraud Detection System

## 🚀 Overview
This project detects fraudulent financial transactions using machine learning and behavioral analysis.

## 🧠 Features
- Fraud probability prediction
- Behavioral analysis
- Transaction anomaly detection
- Explainable AI (reasons for fraud)

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run

```bash
pip install -r requirements.txt
python main.py
